# Sistema de Qualidade QA_AIOps 

Este repositório contém os entregáveis do Trabalho 4, que consiste em um pipeline completo de testes de software e monitoramento preditivo.

## 🛠️ Tecnologias Utilizadas
* **Fase 1 (Teste Funcional):** Java, JUnit 5 e Selenium WebDriver.
* **Fase 2 (Teste de Carga):** Apache JMeter com plugin WebDriver Sampler.
* **Fases 3 e 4 (Machine Learning):** Python, Pandas, Scikit-learn (Isolation Forest/DBSCAN).
* **Fase 5 (Dashboard):** Python (Streamlit/FastAPI).

## Estrutura do Projeto
* `/Fase_1_Funcional_Selenium`: Scripts de automação do cadastro.
* `/Fase_2_Carga_JMeter`: Plano de teste de carga simulando 50 usuários.
* `/Fase_3_4_IA_Colab`: Notebook com o treinamento do modelo preditivo.
* `/Fase_5_Dashboard`: Aplicação web para monitoramento em tempo real.

## Aplicação Alvo
Os testes foram executados sobre o ambiente de desenvolvimento financeiro: https://dev-finance.netlify.app.
